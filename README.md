# openjdk-docker

Only build openjdk8 with debian trixie version. see [Dockerfile](https://github.com/dyrnq/openjdk-docker/blob/main/8/jdk/debian/Dockerfile.hotspot.releases.full)

e.g.

```bash
docker pull adoptopenjdk/openjdk8:jdk8u452-b09-debian
docker pull dyrnq/openjdk8:jdk8u452-b09-debian
```
